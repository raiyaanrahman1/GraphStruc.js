GraphStruc.js

An easy, lightweight, javascript graph library

A graph abstract data type is a network structure in computer science that consists of vertices (or nodes) represented by circles, and lines or arrows connecting different vertices called edges. Graphs are good for visualizing relationships between different objects. My library will allow developers to create animated graph structures and integrate them into their web apps easily. This will be very useful for developers, as coding a graph yourself would be tricky and time-consuming, especially for a complex graph. My graph library is currently responsive and supports draggable vertices.
