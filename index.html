GraphStruc.js

An easy, lightweight, javascript graph library

Made by Raiyaan Rahman

A graph abstract data type is a network structure in computer science that consists of vertices (or nodes) represented by circles, and lines or arrows connecting different vertices called edges. Graphs are good for visualizing relationships between different objects. My library will allow developers to create animated, directed (edges are unidirectional) or undirected (edges are bidriectional) graph structures and integrate them into their web apps easily. This will be very useful for developers, as coding a graph yourself would be tricky and time-consuming, especially for a complex graph.

What makes my library unique is that you can add vertices without specifying a position on the graph and the vertex will automatically be added to a sensible position. Additionally, my library automatically detects if an edge will cause an intersection with an unintended vertex, and curve the edge to reduce the likelihood of an intersection. It does not guarantee that the edge will not intersect, due to absolutely positioned vertexes. Also, developers can animate a particular path in the graph easily, and they can customize features of vertices like their diameter, colour, and content inside, which can be any HTML. Finally, the user interaction to reposition vertices by dragging them is built into the library, and developers can easily give users more freedom and flexibility by giving them access to the libraries methods through the use of forms, as shown in the playground demonstration below.
